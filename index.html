📖 Instrucciones de Uso

1. 🎯 Promociones

  • Selecciona una fecha o mes
  • Busca por banco (ej: Macro, Galicia, Brubank)
  • Verás todas las promociones activas
  • Consulta el descuento, tope mensual y detalles

2. 📄 Cargar PDF

  • Sube el PDF con promociones del mes
  • Selecciona el mes correspondiente
  • Hace backup en la nube (Supabase)

3. 📋 Cierre de Turno

  • Carga todas las ventas por combustible
  • Ingresa los lotes de tarjetas (máx 5)
  • Registra egresos: efectivo, depósitos, medios virtuales, etc
  • Carga los cobros totales
  • El sistema calcula automáticamente si cuadra
  • Guarda todo en la nube

4. 💰 Calculadora de Tope

  • Ingresa el tope de descuento del cliente
  • Ingresa el porcentaje de descuento
  • La app calcula cuánto debe consumir para maximizar el descuento

5. 🔧 Requisitos

  • Funciona en PC, tablet y celular
  • Necesita conexión a internet para sincronizar con Supabase
  • Los datos se guardan automáticamente en la nube
